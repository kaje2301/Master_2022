How to open the data from the minOI bucket: 

# import tools you need 
import s3fs
import xarray as xr

# open the webside with the bucket 
fsg = s3fs.S3FileSystem(anon=False,
      client_kwargs={
         'endpoint_url': 'https://karen.uiogeo-apps.sigma2.no'
      })

# chose which map you want to look into --> here 'velocity-u'
fsg.ls('velocity-u')

#to open everything 
# NB!!! rewrite to zarr, netcdf to slow! 
data_path =  's3://data/CREG12.L75-REF08_mesh_zgr.zarr' #making a path with all the files 
# with zarr:
store = s3fs.S3Map(root=data_path, s3=fsg, check=False)
ds = xr.open_zarr(store=store, consolidated=True)
